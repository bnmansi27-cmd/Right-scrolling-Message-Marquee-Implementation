# Right-scrolling-Message-Marquee-Implementation
This project demonstrates how to marquee right scrolling message using ESP32 microcontroller with CLCD (character / customized liquid character display).
Features:
CLCD interface
Simple and easy-to-use code
Hardware requirements:
Microcontroller (i.e., ESP32 DEV WROOM kit)
CLCD (16X2)
Software Requirements:
Liquid Crystal library
